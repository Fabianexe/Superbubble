LSD.partition¶

LSD.partition.create_auxiliary_graph(c, g)[source]¶

Create the auxiliary graph that have the same superbubbles than g. This function also connects every source to a and every sink to b. This have the effect that every graph have exactly one source (a) and one sink (b). So that the computation of outChild and outParent are much easier.
